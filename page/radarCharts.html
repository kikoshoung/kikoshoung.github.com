Radarcharts （雷达表）

示例1：

		
var analyzer1 = new Radarcharts({
	renderTo: $("#container1"),
	size: "m",
	items: data1,
	maxValue: 100,
	config: { // optional
		title: {
			font: "bold 12px arial",
			color: "red"
		},
		bg: {
			colors: ["red", "green", "blue"],
			num: 3,
			type: 0
		}
	}
});
